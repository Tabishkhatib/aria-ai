Features
Everything a company needs
Built for real enterprise use — not a demo. Every feature is production-ready and fully customisable.

RAG knowledge base

Upload PDFs, Word docs, Excel sheets and CSVs. Documents are chunked, embedded and stored in ChromaDB. The AI retrieves only relevant context per question.

Auth & user management

Secure login and registration with session management. First account becomes admin automatically. Role-based access control built in.

Full white-label branding

Custom assistant name, company name and brand colour. Looks like the client's own product — not a third-party tool. Change everything from the settings panel.

Admin usage dashboard

Track messages, token usage, team members and 14-day activity charts. Admins see everything, employees just chat.

Persistent chat history

Every conversation is saved per user and persists across sessions. Load, continue or delete any past conversation.

Dark mode built in

Automatic light and dark mode based on system preference. No configuration needed — it just works.

How RAG works
Smarter than sending the whole document
RAG means the AI never needs to read 500 pages. It finds the 3 most relevant paragraphs and answers from those.
Without RAG

Send 500 pages to the AI every message. Slow, expensive, hits token limits.

With RAG

Send only the 3 most relevant paragraphs. Fast, cheap, scales to hundreds of docs.

1

Upload documents

PDF, Word, Excel, CSV — any format. Gemini Embedding 001 converts each chunk into a vector.

2

Stored in ChromaDB

Vectors are stored permanently in a local ChromaDB vector database. One time cost per document.

3

Question asked

The question is converted to a vector and compared against all stored vectors to find the best matches.

4

Gemini answers

Only the top 5 most relevant chunks are sent to Gemini 2.5 Flash. Accurate, fast and cost-efficient.
